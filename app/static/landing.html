Đơn giản & hiệu quả

Chỉ 3 bước để bắt đầu giám sát môi trường của bạn ngay hôm nay.

1

Kết nối cảm biến

ESP32 + DHT11 đọc dữ liệu môi trường và gửi lên server mỗi 2 giây qua WiFi.

2

Server xử lý

FastAPI backend nhận và lưu trữ dữ liệu, tự động phát hiện bất thường.

3

Dashboard hiển thị

Giao diện web cập nhật real-time. Xem bất cứ lúc nào trên trình duyệt.
