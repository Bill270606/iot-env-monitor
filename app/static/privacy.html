Privacy Policy

Last updated: June 2026

What we collect

EnvMonitor is a school project for IoT environmental monitoring. The data we handle is limited to:

  • Sensor readings (temperature, humidity, light, soil, rain, UV) from connected ESP32 devices
  • A display name and email entered at login, used only to personalize the dashboard

How it's stored

Account info, theme preference, alert history and chat history are stored locally in your browser (localStorage) — not in a database. Sensor history is kept in server memory for charting and is not permanently archived.

Third parties

If the AI chat assistant is enabled, your messages are sent to Anthropic's Claude API to generate a response. No other third-party services receive your data.

Contact

This is an educational project. Questions can be sent via the GitHub repository.
